<h1>
  Tic-Tac-Toe
</h1>

<p>
  Este repositorio contiene la recreación del clásico juego Tic Tac Toe (Tres en Raya), desarrollado como proyecto final del curso <b>Fundamentos de Python 1</b> de <b>Cisco Networking Academy</b>. Aplica conceptos básicos de Python como estructuras de control, funciones y lógica de programación.
</p>

<h2>
  Aprendizaje
</h2>

<ul>
  <li>Introducción a Python y la Programación</li>
  <li>Tipos de datos, variables y E/S básicas</li>
  <li>Flujo de control y listas</li>
  <li>Colecciones, funciones y excepciones</li>
</ul>
